# AI Resume Screening System

An AI-powered Resume Screening System that analyzes resumes and matches them against job descriptions using NLP and Gemini AI.

## Features

- Resume Upload (PDF/DOCX)
- Job Description Analysis
- Resume Matching Score
- Skill Extraction
- AI-Powered Candidate Evaluation
- User-Friendly Streamlit Interface

## Tech Stack

- Python
- Streamlit
- Gemini AI
- NLP
- PDF Processing

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py# AI Resume Screening System
